#🎵 Music App Fullstack
Aplicación fullstack orientada a la gestión y visualización de canciones, desarrollada con React, Node.js, Express y PostgreSQL.
---
## Tecnologias
React · Node.js · Express · PostgreSQL · JavaScript · CSS3 · REST API · Git · GitHub

## Descripcion
El proyecto implementa una arquitectura separada entre frontend y backend, consumo de API REST, renderizado dinámico de contenido y estructura modular basada en componentes reutilizables.

## Features
- CRUD completo de canciones
- Integración frontend/backend
- Consumo de API REST
- Renderizado dinámico en React
- Componentización reutilizable
- Diseño responsive basado en cards
